# HR Analytics Dashboard 📊

This project is a Power BI dashboard focused on analyzing employee attrition and workforce insights.

## 🔍 Key Features
- Employee overview (KPIs)
- Attrition analysis by department, age, and experience
- Salary band impact on attrition
- Work-life balance & job satisfaction insights

## 📊 Dashboard Pages
1. HR Overview
2. Attrition Insights
3. Key Insights & Recommendations

## 🧠 Key Insights
- Employees in the low salary band show the highest attrition
- Poor work-life balance increases employee exits
- Early tenure employees are more likely to leave
- Long commute distance contributes to attrition

## 💡 Recommendations
- Improve compensation for low salary employees
- Focus on work-life balance initiatives
- Enhance onboarding & early engagement
- Provide remote/flexible work options

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
